# Sinatra::Gyazo

Gyazo for sinatra

## Installation

Add this line to your application's Gemfile:

```ruby
gem 'sinatra-gyazo'
```

And then execute:

    $ bundle

Or install it yourself as:

    $ gem install sinatra-gyazo
